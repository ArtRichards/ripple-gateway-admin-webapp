REST API Reference

Users
A user can register or an admin can list/create users

Resource Description
GET /api/v1/users List all users at this gateway.
POST /api/v1/users Create a new user at this gateway.

Balances
Resource to check on user balances in various currencies with the gateway

Resource Description
GET /api/v1/balances Get a list of all balances for all users
GET /api/v1/users/:userId/balances Get the user's current balances at the gateway

Bank Accounts
Create payment buttons, pages, and iFrames to accept bitcoin on your website.

Resource Description
POST /api/v1/buttons Create a new payment button, page, or iFrame.
POST /api/v1/buttons/:code/create_order Create an order for this button.

Ripple Addresses
List Ripple Addresses related to User accounts.

Resource Description
POST /api/v1/buys Purchase bitcoin by debiting your U.S. bank account.

Ripple Transactions
List transactions made on ripple network to the gateway's address.

Resource Description
GET /api/v1/ripple_transactions Get all rippple transactions

Bank Transactions
List bank transactions made at this gateway

Resource Description
GET /api/v1/bank_transactions Show all bank transactions
GET /api/v1/currencies/exchange_rates Show exchange rates between BTC and other currencies.

Sessions
List merchant orders received.

Resource Description
GET /api/v1/session Get the currently authenticated user's session.
POST /api/v1/session Create a session / sign in.

Withdrawals
List Bank Transaction Withdrawals

Resource Description
GET /api/v1/withdrawals List all withdrawals
GET /api/v1/users/:userId/withdrawals Get a list of a single user's withdrawals.
POST /api/v1/withdrawals As an admin create a withdrawal

Deposits
List Bank Transaction Deposits

Resource Description
GET /api/v1/deposits List all deposits.
GET /api/v1/users/:userId/deposits Show an individual user's deposits.
